Hier..

Ancien imprimeur numérique, graphiste, je me suis reconvertie dans le dévellopement Web afin de nourrir mon amour pour les nouvelles technologie et une envie de travailler pour un métier ou l’apprentissage est paermanent ! Je vous propose mes services à la fois techniques et graphiques, passant d’un logiciel Adobe a VS Code, afin de m’adapter à la fois a votre identité visuel et aux technologies pour la gestion de votre entreprise.

Aujourd'hui

Je suis donc, suite à ma reconversion, en formation de Développeur Web/Web mobile a l’AFPA Cambrai, depuis Juin 2022.

Demain

Une envie et une curiositée me poussant a evoluer et a apprendre.

HTML 5 / CSS 3 / BOOTSTRAP 5

Afin de fournir un front performant et totalement reponsive, j'utilise principalement le framework Bootstrap 5. Je développe directement sur Vs-Code sans passer par un CMS afin de fournir des pages légères avec un référencement optimal.

JAVASCRIPT / REACT

Pour encore plus de performance, je peux développer votre site en one page via le modèle MCV, le tout en Javascript et React, une technologie utilisé et développé par Facebook, Instagram.

MY SQL / PHP

Pour des sites dynamiques avec une base de données. La mise à dispo d'un back office comprenant des outils pour la gestion de votre site via un status d'administrateur !

Image / Logo / 3D / Video / Photo

En plus de réaliser votre site web, je peux également m'occuper de votre logo, votre charte graphique, mais pas que... De la création d'une capsule vidéo, une campagne de e-mailing, une affiche. Amateur en photo, je peux également faire le shooting de vos produits, ainsi que leur retouche sur LightRoom et Photoshop. De ce fait, je peux vous fournir un site web complet, vous n'avez rien à faire, je m'occupe de tout !
